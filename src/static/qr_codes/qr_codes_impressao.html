Instruções para uso:

  • Coloque um QR Code em cada mesa do restaurante
  • Os clientes devem escanear o QR Code com a câmera do celular
  • O sistema irá abrir automaticamente no navegador
  • O cliente poderá fazer o pedido diretamente pelo celular
  • Os pedidos aparecerão no painel administrativo em tempo real

Acesso ao Painel Administrativo:

URL: http://localhost:5000/admin

Use o painel para gerenciar mesas e confirmar pagamentos
