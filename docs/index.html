Sistema de Restaurante QR Code

Um projeto de demonstração para portfólio

Visão Geral do Projeto

Este projeto é um sistema web local completo, desenvolvido para simular o funcionamento de um restaurante moderno que utiliza QR Codes nas mesas. O objetivo principal é demonstrar habilidades em desenvolvimento full-stack, cobrindo desde o backend com Python/Flask até o frontend responsivo com HTML, CSS e JavaScript.

O sistema permite que os clientes acessem o cardápio, façam pedidos diretamente de seus celulares e que o restaurante receba e gerencie esses pedidos através de um painel administrativo.

Funcionalidades Principais

QR Code nas Mesas

Geração automática de QR Codes únicos para cada mesa, direcionando o cliente diretamente para o cardápio digital.

Cardápio Digital Responsivo

Interface mobile-first para o cliente, permitindo fácil navegação pelo cardápio e adição de itens ao carrinho.

Sistema de Pedidos

Clientes podem montar seus pedidos, enviá-los para a cozinha e solicitar o fechamento da conta.

Painel Administrativo

Gerenciamento em tempo real do status das mesas, visualização de pedidos e confirmação de pagamentos.

Armazenamento Simplificado

Utilização de SQLite para persistência de dados de mesas, cardápio e pedidos, ideal para demonstração.

Código Organizado

Estrutura de projeto clara com separação de responsabilidades entre frontend e backend (Flask).

Tecnologias Utilizadas

Python Flask SQLite HTML5 CSS3 JavaScript qrcode (Python)

Notas de Segurança e Melhorias Futuras

É importante ressaltar que este projeto foi desenvolvido com foco em demonstração de funcionalidades e não em segurança para ambiente de produção. O README.md completo do projeto (disponível no repositório) detalha as melhorias de segurança necessárias para um ambiente real, como autenticação, autorização, validação de entrada e criptografia de dados.

Entre as melhorias futuras planejadas, destacam-se a integração com sistemas de pagamento, notificações em tempo real via WebSockets e a implementação de testes automatizados.
